🔐 Email OTP Verification Test

Step 1: Sign Up with Email

Step 2: Check System Logs

When you sign up, the backend will:

  • Generate a random 6-digit OTP
  • Save OTP to database with 10-minute expiry
  • Send OTP to your email via Gmail SMTP

📧 OTP Location: Check your email inbox (or spam folder) for the verification code

🔍 Backend Logs: Look for "✅ Email sent:" message in terminal

Step 3: Verify Email with OTP

After signing up, you'll receive a 6-digit code in your email

Step 4: Resend OTP

If you didn't receive the OTP, click to resend:

📋 API Endpoints Summary

Endpoint Method Purpose Authentication
/auth/signup POST Register new user & send email OTP None
/auth/send-email-otp POST Resend email OTP JWT Token
/auth/verify-email-otp POST Verify OTP and mark email as verified JWT Token
