🆘 COMMON ISSUES & QUICK FIXES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CORS ERROR
→ Add your Vercel URLs to backend CORS in Railway env vars

404 ON API ENDPOINT
→ Check backend logs (Railway dashboard)
→ Verify routes are imported in server.js

FIRESTORE NOT WORKING
→ Check Firebase credentials in Railway
→ Verify private key has literal \n for newlines

PAYMENT ERROR
→ Use production Razorpay keys (rzp_live_)
→ NOT test keys (rzp_test_)

BLANK PAGE / BUILD FAILED
→ Check Vercel build logs
→ Verify root directory is set correctly

DATABASE EMPTY
→ Go to Firestore console
→ Create database if not exists
→ Add test data via API

CAN'T LOGIN
→ Check JWT_SECRET in Railway env
→ Verify ADMIN_EMAIL and ADMIN_PASSWORD

ROLLBACK QUICKLY (if critical issue)
→ git revert HEAD
→ git push origin main
→ Wait 5 minutes for auto-redeploy
